# Writing style

## Voice

- Third person, active voice.
- Short sentences. No run-ons.
- No filler phrases ("it is important to note", "as we can see").

## Structure

- Lead with the conclusion, then the evidence.
- One idea per paragraph.
- Prefer bullet lists for parallel items.

## Claims

- Every non-obvious claim cites which prior step it comes from.
- If a claim is uncertain, say so explicitly — do not smuggle hedge words into otherwise assertive prose.

## Length

- Shorter is better. If a section can be 100 words instead of 300, make it 100.
